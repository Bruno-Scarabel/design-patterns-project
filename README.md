# Padrões de projeto - Exemplo prático (Design Patterns)

Este repositório apresenta **três padrões de projeto** clássicos, divididos em suas três categorias principais:

- **Comportamental:** Observer  
- **Criacional:** Factory Method  
- **Estrutural:** Decorator  

Os exemplos e explicações são baseados no catálogo do [Refactoring Guru](https://refactoring.guru/pt-br/design-patterns).  
Todo o código e documentação deste repositório foram criados com o auxílio do **ChatGPT (GPT-5)**, conforme solicitado no enunciado.
